### Prerequisites

* Node.js 16+ installed
  Check your version with:

  ```bash
  node -v
  ```

---

### Local Installation

1. Clone the repository:

   ```bash
   git clone <REPOSITORY_URL>
   ```

2. Navigate to the project folder:

   ```bash
   cd AdventureTime-OnlineGame
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the server:

   ```bash
   npm start
   ```

5. Open your browser and go to:

   ```
   http://localhost:3000
   ```
